## Quick Start

1. **Clone and Install**
   ```bash
   npm create next-app@latest leetcode-75-tracker
   cd leetcode-75-tracker
   npm install
   ```

2. **Copy the Code**
   Replace the generated files with the code from this artifact.

3. **Run Development Server**
   ```bash
   npm run dev
   ```

4. **Deploy to Vercel**
   ```bash
   npm run build
   vercel --prod
   ```
